Introduction

My Shiny App is hosted at http://konnectome.shinyapps.io/DDP_shiny_stocks.

The App provides a dynamic and interactive way to retrieve and study historical stock prices on demand via R package quantmod.

It provides the following capabilities: 1. allows the user to study the prices at different time frequencies such as Daily, Weekly, and Monthly, along with data on trading volume for each period of chosen frequency. 2. allows the user to specify length of historical periods to examine via a slider widget in Shiny. Default length is set at 50 periods with a maximum length of 100 periods. 3. allows the user to examine price action with respect to its trending, momentum, and volatity characteristics by providing dynamic Technical Analysis indicators. 4. allows the user to choose logrithmic scale to display the price to detect trends more effectively. 5. allows user to choose from 3 stocks.

Implemented indicators include: Bollinger Band for volatity; MACD (Moving Average Convergence Divergence) for trending; RSI (Relative Strength Indicator) for momentum; SMI (Stochastic Momentum Indicator) for momentum.

Conclusion

This App can be extended in a number of different directions to become a full dashboard for data mining stock price patterns.

In addition to current implementation of listing 3 stocks, the App can either list many more stocks and/or allow user to dynamically input a symbol of a stock of particular interest.

Currently, the historical prices are dynamically downloaded via quantmod each time the App is run. We can store historical prices of a much broader universe of stocks in a database and access dynamically from this database.

In addition to allowing only 4 indicators, the App can implement many other indicators avaialbe via R packages quantmod and TTR.

Lastly, in addition to indicators available in R packages, we can implement our own machine learning algorithms for data mining and predictive analytics with a fully functional back-testing engine.
